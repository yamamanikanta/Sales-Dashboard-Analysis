# Sales-Dashboard-Analysis

Objective: Analyze sales performance across different product categories and provide interactive month-by-month and productwise insights for better decision-making.

Tools & Technologies: Microsoft Excel, Pivot Tables, Pivot Charts, Data Visualization, Dashboard Design, Data Analysis

• Analyzed sales data across multiple product categories to identify trends, patterns and performance insights.

• Created interactive Excel dashboards with user controls and filters for category-wise, monthly and product-wise analysis.

• Developed charts, graphs, and visual summaries to present complex sales data in an easy-to-understand format.

• Applied data formatting and visualization techniques to improve dashboard usability and reporting clarity.

Result: Enabled efficient sales monitoring and trend analysis, helping users quickly compare product category performance and make data-driven business decisions.
